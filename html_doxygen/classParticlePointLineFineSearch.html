Detailed Description

template<int dim>
class ParticlePointLineFineSearch< dim >

This class is used for fine particle-point and particle-line contact search. Fine search is used to find all the particles which are physically in contact with boundary lines and points, and obtain all the required information for calculation of the corresponding contact force

Iterates over a map of tuples (particle_line_contact_candidates) to see if the particle-line pairs are in contact or not. If they are in contact, the normal overlap, normal vector of contact and contact normal relative velocity are stored in a map which is the output of this function
