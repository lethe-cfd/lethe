Detailed Description

template<int dim>
class GLSNavierStokesAssemblerCore< dim >

Class that assembles the core of the Navier-Stokes equation. This class assembles the weak form of: $$\mathbf{u} \cdot \nabla \mathbf{u} - \nabla p - \nu \nabla^2 \mathbf{u} =0 $$ with a full GLS stabilization including the laplacian of the test function.
