Detailed Description

template<int dim>
class HeatTransferAssemblerCore< dim >

Class that assembles the core of the heat transfer equation. This class assembles the weak form of: \( - k * \nabla^2 T + \rho * cp * \mathbf{u} * \nabla T - f - \tau : \nabla \mathbf{u} =0 \).

Template Parameters
dimAn integer that denotes the number of spatial dimensions
Parameters
simulation_controlShared pointer of the SimulationControl object controlling the current simulation
