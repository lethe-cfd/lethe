Detailed Description

template<int dim>
class GLSIsothermalCompressibleNavierStokesAssemblerCore< dim >

Class that assembles the core of the isothermal compressible Navier-Stokes equations. According to the following weak form: with a full GLS stabilization including the laplacian of the test function.
