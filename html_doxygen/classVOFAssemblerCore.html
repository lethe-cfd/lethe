Detailed Description

template<int dim>
class VOFAssemblerCore< dim >

Class that assembles the core of the VOF solver. This class assembles the weak form of: $$\mathbf{u} \cdot T \cdot \nabla T =0 $$ with an SUPG stabilization.

Template Parameters
dimAn integer that denotes the number of spatial dimensions
