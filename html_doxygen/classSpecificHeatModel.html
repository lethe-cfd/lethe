Detailed Description

Abstract class that allows to calculate the specific heat on each quadrature point using the temperature of the fluid. magnitude. SpecficiHeat::get_specific_heat() is a pure virtual method, since it can only be calculated knowing the model for the specific heat that has been specifid.
