Detailed Description

template<int dim>
class CahnHilliardAssemblerCore< dim >

Class that assembles the core of the Cahn-Hilliard equation. This class assembles the weak form of: dPhi/dt + u * gradPhi = div(M(Phi)*grad eta) eta - f(Phi) + epsilon^2 * div(grad Phi) = 0.
