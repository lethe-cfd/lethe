Detailed Description

template<int dim>
class VorticityPostprocessor< dim >

used to calculate the vorticity field within the domain. The vorticity is defined as $\mathbf{\omega} = \nabla \times \mathbf{u}$ or, in other notation, $\mathbf{\omega} = rot(\mathbf{u})$.
