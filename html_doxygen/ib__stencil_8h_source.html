47  /**
48  * @brief
49  * Define the points for the IB stencil, based on the order and the particle
50  * position as well as the DOF position Depending on the order, the output
51  * variable "point" change definition. In the case of stencil orders 1 to 4
52  * the variable point returns the position of the DOF directly. In the case of
53  * high order stencil, it returns the position of the point that is on the IB.

55  * cell used for the stencil definition and the locations of the points use in
56  * the stencil calculation.
57  *
58  * @param order, the stencil order.
59  * @param length_ratio the length ratio between the 2 side of the stencil.
60  * @param p, the IB particle that cuts the cell.
61  * @param dof_point, the support point of the DOF.
62  * @param cell_guess A guess of the cell containing the evaluation point, which
63  * is useful to reduce computation time
64  */
