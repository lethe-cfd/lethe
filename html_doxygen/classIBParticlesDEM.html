Detailed Description

template<int dim>
class IBParticlesDEM< dim >

A solver class for the DEM used in conjunction with IB particles and gls_sharp_navier_stokes. This class defines and uses some functions of the DEM class that have been modified and simplified to be compatible with IB_particles.

Template Parameters
dimAn integer that denotes the dimension of the space in which the flow is solved
