The ListInsertion class inserts particles using a list specific position. This allows the insertion of any number of particles at a well-controled location which is especially useful from a testing perspective. The code ensures that the number of positions provided in the x,y (and possibly z) direction is coherent. If more particles than the number of position in the list are requested, the class will continue inserting particles at the insertion frequency using the list of position. There is no mechanism in place that prevents the overlap of these new particles with previous ones.

Parameters
particle_handlerThe particle handler of particles which are being inserted
triangulationTriangulation to access the cells in which the particles are inserted
dem_parametersDEM parameters declared in the .prm file
