Detailed Description

template<int dim>
class ExplicitEulerIntegrator< dim >

Implementation of a classical explicit euler scheme for the integration of the particle motion. Note that reinitilization of force and torque is also integrated into integration class

Note
Euler is a first-order integration scheme. Calculation proceudre:

x(n+1) = x(n) + v(n) * dt v(n+1) = v(n) + a(n) * dt a(n+1) = F(n+1) / m

Author
Shahab Golshan, Bruno Blais, Polytechnique Montreal 2019-
