Detailed Description

template<int dim>
class CahnHilliardAssemblerBDF< dim >

Class that assembles the transient time arising from BDF time integration for the Cahn-Hilliard equations. For example, if a BDF1 scheme is chosen, the following is assembled \(\frac{\mathbf{T}^{t+\Delta t}-\mathbf{T}^{t}}{\Delta t}\).

Template Parameters
dimAn integer that denotes the number of spatial dimensions
