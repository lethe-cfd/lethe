Detailed Description

template<int dim>
class HeatTransferAssemblerLaserExponentialDecayVOF< dim >

Class that assembles the laser heating as a volumetric source for the heat transfer solver when VOF is enabled. Exponentially decaying model is used to simulate the laser heat source: "Liu, S., Zhu, H., Peng, G., Yin, J. and Zeng, X., 2018. Microstructure prediction of selective laser melting AlSi10Mg using finite element analysis. Materials & Design, 142, pp.319-328." The laser heat source is only applied in the metal (when phase value is non-null) using the phase value alpha as a multiplying factor on the laser heat source.
