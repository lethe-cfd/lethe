Detailed Description

template<int dim, int spacedim = dim>
class GLSNitscheNavierStokesSolver< dim, spacedim >

A solver class for the Navier-Stokes equation using GLS stabilization and Nitsche immersed boundary method

Template Parameters
dimAn integer that denotes the dimension of the space in which the flow is solved
