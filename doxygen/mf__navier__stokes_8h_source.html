241 /**
242  * @brief A solver for the incompressible Navier-Stokes equations implemented
243  * in a matrix-free fashion.
244  *
245  * A matrix-free stabilized solver for the incompressible Navier-Stokes
246  * equations implemented in a matrix-free fashion with a sum-factorization
247  * approach. It uses a continuous Galerkin discretization and solves the
248  * fully-coupled discretized problem in a monolithic way using multigrid
249  * preconditioners.
250  *
251  * @tparam dim An integer that denotes the dimension of the space in which
252  * the flow is solved.
253  */
