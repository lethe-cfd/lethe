201 /**
202  * @brief A solver for the incompressible Navier-Stokes equations implemented
203  * in a matrix-free fashion.
204  *
205  * A matrix-free stabilized solver for the incompressible Navier-Stokes
206  * equations implemented in a matrix-free fashion with a sum-factorization
207  * approach. It uses a continuous Galerkin discretization and solves the
208  * fully-coupled discretized problem in a monolithic way using multigrid
209  * preconditioners.

212  * the flow is solved.
