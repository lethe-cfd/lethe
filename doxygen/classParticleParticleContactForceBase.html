Detailed Description

template<int dim>
class ParticleParticleContactForceBase< dim >

Base class for the particle-particle contact force models This class does not implement any of the models, but ensures that an interface without template specialization is available. All of the actual implementation of the models are carried out in the ParticleParticleContactForce class which is templated by the contact model type.
