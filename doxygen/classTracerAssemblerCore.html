Detailed Description

template<int dim>
class TracerAssemblerCore< dim >

Class that assembles the core of the Tracer equation. This class assembles the weak form of: \(\mathbf{u} \cdot \nabla T - D \nabla^2 =0 \) with an SUPG stabilization.

Template Parameters
