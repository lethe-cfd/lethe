Detailed Description

template<int dim>
class GLSIsothermalCompressibleNavierStokesVOFAssemblerCore< dim >

Class that assembles the core of the Navier-Stokes equation with free surface using VOF modeling. According to the following weak form: \(\nabla \cdot (\rho \mathbf{u}) + \rho \mathbf{u} \cdot \nabla \mathbf{u} - \nabla p - \mu \nabla\cdot (\nabla \mathbf{u} + (\nabla \mathbf{u})^T) = 0 \) with a full GLS stabilization.
