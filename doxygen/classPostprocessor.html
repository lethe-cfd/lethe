Postprocessor Class Reference

Detailed Description

local heat fluxes \((q = -k * \nabla T)\) of a material at every evaluation point of the domain.

Parameters
p_thermal_conductivity_modelThermal conductivity model of the material
p_material_stringType of material: "f" (fluid) or "s" (solid)
p_idEither fluid ID or solid ID (as initialized in the physical properties) used to name the postprocessed variable vector.
p_mesh_material_idID corresponding to the material as identified in the mesh. By convention, fluids are given the material_id 0 and solids follow with ids 1, 2, etc. This is used to identify which cell lies in which material.
p_solid_phase_presentBoolean indicating if the simulation has solids. If there is no solid, the material_id of cells is not checked.
