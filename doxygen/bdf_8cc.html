Vector<double> bdf_coefficients ( const Parameters::SimulationControl::TimeSteppingMethod  method,
const std::vector< double >  time_steps 
)

Calculate the coefficients required for BDF integration of order n. The coefficients are determined through a recursion algorithm. The algorithm is taken from : Hay, Alexander, et al. "hp-Adaptive time integration based on the BDF for viscous flows." Journal of Computational Physics 291 (2015): 151-176.

Parameters
methodThe time stepping method
time_stepsa vector containing all the time steps. The time steps should be in reverse order. For example, if the method is a BDF2, it uses three values for the time : (t) (t-dt_1) and (t-dt_2). Thus the time step vector should contain dt_1 and dt_2.
