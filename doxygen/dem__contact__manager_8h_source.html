43  * @brief Manage the numerous of contact detection search operations in the DEM.

52  /**
53  * @brief Execute functions to find the lists of cell neighbors of the active
54  * cells in the triangulation.
55  *
56  * For contacts of particles, the neighbor search excludes the repetitions of
57  * neighbors: B is neighbor of A but A will not be neighbor of B. For contacts
58  * of particles with floating meshes, the neighbor search includes all
59  * neighbors. These floating mesh contacts searches need all the particles
60  * located in all the neighbor cells of the main cell to detect
61  * collisions with the floating mesh.
62  *
63  * @param[in] triangulation The triangulation to access the information of the
64  * cells.
65  * @param[in] periodic_boundaries_cells_information Information of periodic
66  * cells used if periodic boundaries are enabled (next parameter).
67  * @param[in] has_periodic_boundaries Allow manipulations of periodic
68  * containers if required.
69  * @param[in] has_floating_mesh Allow the computation of full neighbor lists
70  * of active cells computation if required.
71  */
72  void
73  execute_cell_neighbors_search(
74  const parallel::distributed::Triangulation<dim> &triangulation,
75  const typename DEM::dem_data_structures<dim>::periodic_boundaries_cells_info
76  periodic_boundaries_cells_information,
77  const bool has_periodic_boundaries = false,
78  const bool has_floating_mesh = false);

116  * contact pairs.

119  * pairs to the contact containers when particles are exchanged between
120  * processors after the fine search.
121  * Contact pairs are local particle-particle, ghost particle-particle,
122  * particle-wall, particle-floating wall contacts and particle-floating mesh
