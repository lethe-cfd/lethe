Detailed Description

template<typename VectorType>
class PhysicsSolver< VectorType >

Class that has all the common elements of a physics solver. It creates the nonlinear solver as specified by the user using the parameters file and provides all the necessary elements needed by the solver to solve a physics problem.

Parameters
non_linear_solver_parametersA set of parameters that will be used to construct the non-linear solver
