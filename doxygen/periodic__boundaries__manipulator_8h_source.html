37 /**
38  * This class corresponds to a manipulator of the particles crossing periodic
39  * cells in DEM. It maps cell information of the pairs of periodic cells
40  * and manipulate the particles location when they cross periodic boundaries,
41  * i.e. when a particle cross a periodic boundary (0 or 1) its location is
42  * modified with the offset between faces of periodic cells on periodic
43  * boundaries.
44  *
45  * @note Currently the code only supports one pair of periodic boundaries and
46  * those boundaries must be parallel and aligned with an axis of the domain
47  * (e.g., x axis).
48  */
49

63  void
64  set_periodic_boundaries_information(
