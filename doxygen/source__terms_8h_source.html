35 /**
36  * The source term class provides an interface for all common
37  * Element required for the introduction of a source term in equations
38  * All equation-specific source term should derive
39  * from the base class but also call it's declare_parameters and
40  *parse_parameters routine. This allows specialize class to focus on their
41  *specificity and forget about other non-specific elements that are generic to
42  *the calculation of analytical solutions
43  **/
